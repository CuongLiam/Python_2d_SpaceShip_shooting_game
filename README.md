# Alien Invasion Ultimate

A 2D space shooter game built with Python and Pygame. Control your spaceship, shoot aliens, and survive as long as possible!

## Features
- Main menu, settings, and pause menu
- Difficulty selection (Easy, Normal, Hard)
- Volume slider for music and sound effects
- Score and high score tracking
- Sound effects and background music

## Controls
- Arrow keys: Move ship left/right
- Space: Fire bullet
- P: Pause/Resume game
- Q: Quit game
- Mouse: Click buttons and adjust volume slider

## Code Structure
- `main.py`: Main game loop, event handling, menu logic
- `settings.py`: Game settings and difficulty management
- `game_stats.py`: Tracks score, level, and lives
- `scoreboard.py`: Displays score, level, and lives
- `button.py`: UI button class
- `slider.py`: UI slider for volume control
- `sprites/ship.py`: Player ship sprite
- `sprites/alien.py`: Alien sprite
- `sprites/bullet.py`: Bullet sprite

## Assets
- Images: `assets/images/`
- Sounds: `assets/sounds/`

## How to Run
1. Install Python 3.x and Pygame (`pip install pygame`)
2. Run `main.py`
